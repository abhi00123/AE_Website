# Abhishek Enterprise
> Premium Tile & Marble Contractors | Specialized Architectural Excellence in Pune

![AE Banner](public/favicon.svg)

Abhishek Enterprise is a premier contracting firm based in Pune, specializing in high-fidelity tile fitting, marble polishing, and sophisticated granite work. This modern, responsive web application showcases our craftsmanship, services, and diverse portfolio of residential and commercial projects.

---

## ✨ Features

- **📱 Fully Responsive**: Optimized for every screen size, from mobile devices to desktop monitors.
- **⚡ Blazing Fast**: Built with React and Vite for near-instant load times and high performance.
- **💎 Premium Design**: High-fidelity architectural aesthetic with smooth transitions and Material Design influences.
- **🛠️ Service Portfolio**: Detailed showcases of Tile Fitting, Marble Polishing, and Granite specialized work.
- **📂 Dynamic Gallery**: A high-resolution masonry gallery to explore our past projects.
- **📞 Quick Contact Actions**: Integrated WhatsApp, direct call, and estimate forms for seamless lead generation.

---

## 🚀 Tech Stack

- **Framework**: [React 18+](https://react.dev/)
- **Build Tool**: [Vite 6+](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 3.4](https://tailwindcss.com/)
- **Routing**: [React Router Dom 6.28](https://reactrouter.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Material Symbols Outlined](https://fonts.google.com/icons)
- **Typography**: [Epilogue](https://fonts.google.com/specimen/Epilogue) (Headlines) & [Inter](https://fonts.google.com/specimen/Inter) (Body)

## 🛠️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm or yarn

### Installation

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   ```

2. **Navigate to the project directory**:
   ```bash
   cd "Abhishek Enterprise"
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Start the development server**:
   ```bash
   npm run dev
   ```

---

## 🏗️ Project Structure

- `src/components`: Reusable UI components (Navbar, Footer, MobileNav, etc.).
- `src/pages`: Main page components (Home, Services, Gallery, Contact, etc.).
- `src/index.css`: Global styles and Tailwind configuration.
- `public/`: Static assets like favicons and images.

---

## 📦 Deployment & Build

To create a production-ready build, run:

```bash
npm run build
```

The optimized application will be located in the `dist` folder.
